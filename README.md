# MovieApp Technologies 
* react js
* tailwind css and css modules
* react query (cache)
* react context api
* firebase (auth)

**Application allowes**
 * to search movies
 * to filter movies
 * create account to build watchList
 * to view movie details (like casts)
 * to view similiar movies
